#Fire_Station_MU

Dieser Mod fügt 2 weiter Feuerwachen hinzu.

das Institut der Feuerwehr:
---------------------
Einsatzkräfte Reichweite ( im Vergleich zum Orginal ) : +8%
Einsatzkräfte ( im Vergleich zum Orginal ) : +300% (4 Einheiten)
Einsatzkräfte Bewegungsrate ( im Vergleich zum Orginal ) : +30%
Einsatzkräfte Löschgeschwindigkeit ( im Vergleich zum Orginal ) : +100%
Verringert Brandgefahr ( im Vergleich zum Orginal ) : +45%
Reichweite Brand-reduzierung ( im Vergleich zum Orginal ) : +60%


die Freiwillige Feuerwehr:
---------------------
Einsatzkräfte Reichweite ( im Vergleich zum Orginal ) : -16%
Einsatzkräfte ( im Vergleich zum Orginal ) : +100% (2 Einheiten)
Einsatzkräfte Bewegungsrate ( im Vergleich zum Orginal ) : +15%
Einsatzkräfte Löschgeschwindigkeit ( im Vergleich zum Orginal ) : +50%
Verringert Brandgefahr ( im Vergleich zum Orginal ) : -3%
Reichweite Brand-reduzierung ( im Vergleich zum Orginal ) : -15%



Die Gebäude wurden in das Orginalbaumenü von der Feuerwache integriert.
