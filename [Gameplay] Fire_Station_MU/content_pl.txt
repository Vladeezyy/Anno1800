#Fire_Station_MU

Ten mod dodaje jeszcze 2 stanowiska strazackie.

instytut strazy pozarnej:
---------------------
Zakres sily reakcji ( w porównaniu do oryginalu ) : +8%
Sily reakcji ( w porównaniu do oryginalu ) : +300% (4 jednostek)
Szybkosc przemieszczania sie sily ( w porównaniu do oryginalu ) : +30%
Szybkosc gaszenia sily ( w porównaniu do oryginalu ) : +100%
Zmniejsza zagrozenie pozarowe ( w porównaniu do oryginalu ) : +45%
Zmniejszenie zasiegu ognia ( w stosunku do oryginalu ) : +60%


ochotnicza straz pozarna:
---------------------
Zakres sily reakcji ( w porównaniu do oryginalu ) : -16%
Zespoly zadaniowe ( w porównaniu do oryginalu ) : +100% (2 szt.)
Szybkosc przemieszczania sie sily ( w porównaniu do oryginalu ) : +15%
Szybkosc gaszenia sily ( w porównaniu do oryginalu ) : +50%
Zmniejsza zagrozenie pozarowe ( w porównaniu do oryginalu ) : -3%
Zmniejszenie zasiegu ognia ( w stosunku do oryginalu ) : -15%



Budynki zostaly wkomponowane w oryginalne menu drzewiaste remizy.
