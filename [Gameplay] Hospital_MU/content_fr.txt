#Hospital_MU

Ce mod ajoute 2 hôpitaux / médecins supplémentaires.

le sanatorium :
---------------------
Portée des forces d'intervention ( par rapport à l'original ) : +8%
Forces d'intervention ( par rapport à l'original ) : +100% ( 4 unités )
Taux de déplacement des unités ( par rapport à l'original ) : +30%
Vitesse d'intervention ( par rapport à l'original ) : +100%
Réduction de la maladie ( par rapport à l'original ) : +45%
Portée de la réduction de la maladie ( par rapport à l'original ) : +60%


des praticiens de la santé :
---------------------
Portée des forces d'intervention ( par rapport à l'original ) : -16%
Forces d'intervention ( par rapport à l'original ) : +50% ( 3 unités )
Taux de déplacement des unités ( par rapport à l'original ) : +15%
Vitesse d'intervention ( par rapport à l'original ) : +50%
Réduction de la maladie ( par rapport à l'original ) : -3%
Portée de la réduction de la maladie ( par rapport à l'original ) : -15%



Les bâtiments ont été intégrés dans le menu d'arbre original de l'hôpital, disponible dans Enbesa.
