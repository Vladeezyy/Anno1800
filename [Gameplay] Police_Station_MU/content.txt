#Police_Station_MU

Dieser Mod fügt 2 weiter Polizeiwachen hinzu.

das Staatsgefängniss:
---------------------
Einsatzkräfte Reichweite ( im Vergleich zum Orginal ) : +8%
Einsatzkräfte ( im Vergleich zum Orginal ) : +300% (4 Einheiten)
Einsatzkräfte Bewegungsrate ( im Vergleich zum Orginal ) : +30%
Einsatzkräfte Vollzugsgeschwindigkeit ( im Vergleich zum Orginal ) : +100%
Verringert Aufstände ( im Vergleich zum Orginal ) : +45%
Reichweite Aufstands-reduzierung ( im Vergleich zum Orginal ) : +60%


die Mobile Einsatztruppe:
---------------------
Einsatzkräfte Reichweite ( im Vergleich zum Orginal ) : -16%
Einsatzkräfte ( im Vergleich zum Orginal ) : +100% (2 Einheiten)
Einsatzkräfte Bewegungsrate ( im Vergleich zum Orginal ) : +15%
Einsatzkräfte Vollzugsgeschwindigkeit ( im Vergleich zum Orginal ) : +50%
Verringert Aufstände ( im Vergleich zum Orginal ) : -3%
Reichweite Aufstands-reduzierung ( im Vergleich zum Orginal ) : -15%



Die Gebäude wurden in das Orginalbaumenü von der Polizeiwache integriert.
